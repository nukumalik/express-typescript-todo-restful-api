### Getting Started

#### Clone Repo

```bash
$ git clone git@github.com:nukumalik/express-typescript-todo-restful-api.git
```

#### Enter into directory

```bash
$ cd express-typescript-todo-restful-api
```

#### Install Packages

```bash
$ npm run install
```

#### Rename and Config ENV

```bash
$ mv .env.example .env
```

#### Run Development

```bash
$ npm run dev
```

#### Build App

```bash
$ npm run build
```

#### Run Server

```bash
$ npm run start
```
